# Агромаштех — Баримт бүртгэл (PWA)

Захирлын ибаримт бүртгэл. Гар утсанд суудаг, **сүлжээгүй ч ажилладаг** апп.
Баримтыг зураг/QR-аар авч хадгалаад, сүлжээ ортол автоматаар Google Sheet + Drive рүү илгээнэ.

---

## 1. Тохиргоо (заавал)

`app.js` файлын дээд талын `CONFIG`-г бөглөнө:

```js
var CONFIG = {
  API_URL:   'https://script.google.com/macros/s/XXXX/exec',  // Apps Script /exec URL
  API_TOKEN: 'AGRO-2026-x7k9',   // backend-ийн API_TOKEN-тэй ЯГ ижил
  ORG_TIN:   '5203449'           // Агромаштех ТТД
};
```

---

## 2. GitHub Pages дээр байрлуулах

1. GitHub дээр шинэ repo үүсгэ (жишээ: `agro-baramt`), **Public**
2. Энэ хавтасны бүх файлыг оруул (icons, lib багтаана)
3. **Settings ▸ Pages ▸ Branch: main / root ▸ Save**
4. 1–2 минутын дараа хаяг гарна:
   `https://<нэр>.github.io/agro-baramt/`

> ⚠️ HTTPS заавал (камер, service worker HTTPS дээр л ажиллана). GitHub Pages өөрөө HTTPS тул асуудалгүй.

---

## 3. Утсанд суулгах

- **iPhone (Safari):** хаягийг нээ ▸ Хуваалцах ▸ "Нүүр дэлгэцэд нэмэх"
- **Android (Chrome):** хаягийг нээ ▸ цэс ▸ "Апп суулгах / Нүүр дэлгэцэд нэмэх"

Суулгасны дараа энгийн апп шиг нээгдэнэ. Эхний удаа онлайнаар нэг нээхэд бүрхүүл кэшлэгдэж, дараа нь **сүлжээгүй ч нээгдэнэ.**

---

## 4. Ашиглах урсгал

1. **＋ Баримт нэмэх**
2. Төрөл сонго: Дараа тооцоо / Дотоод томилолт / Гадаад томилолт
3. **Зураг дар** эсвэл **QR уншуул**
4. **Зорилго** бич (жишээ: "РСМ зочдын өдрийн хоол")
5. **Хадгалах**

Сүлжээгүй бол дараалалд хадгалагдаад, сүлжээ ортол автоматаар илгээгдэнэ.
Нүүр дэлгэц дээр "Хүлээгдэж буй" тоо болон "Илгээгдсэн / Алдаа" төлөв харагдана.

---

## 5. Файлын бүтэц

```
agro-baramt/
├─ index.html        UI
├─ app.js            логик (камер, QR, offline дараалал, sync)
├─ sw.js             service worker (offline бүрхүүл)
├─ manifest.json     PWA тохиргоо
├─ lib/
│  └─ html5-qrcode.min.js   (offline QR — CDN-гүй)
├─ icons/            аппын дүрснүүд
└─ README.md
```

## Тэмдэглэл

- `sw.js` доторх файлыг өөрчилсөн бол `CACHE = 'agro-baramt-v1'` → `v2` болгож ахиулна (кэш шинэчлэгдэнэ).
- QR задлалт нь eBarimt-ийн хувилбараас хамаарна — бодит баримт дээр туршиж нарийсгана.
- Backend: `01_Setup.gs`, `02_WebApp.gs` (Google Apps Script).
